# `Endpass Faucet`

Faucet for getting test ETH in one click

## Usage

```sh
import FaucetComponents from '@endpass/faucet';

...

Vue.use(FaucetComponents);

```

## Available Components
### Components

* VFaucetButton

## Install

```sh
yarn add @endpass/faucet
```

## Contribute

```sh
git clone https://github.com/endpass/endpass-core
cd packages/faucet
yarn
```

### Build

```sh
yarn build
```

or

```sh
yarn build-watch
```

### Running Tests

```sh
yarn unit
```
